# 大模型应用后端教材-总纲

## 教材定位

这套教材面向传统 Go/Java 后端工程师，目标是转向大模型应用后端、Agent 平台后端、RAG 工程、AI Infra 或大模型产品工程岗位。

本教材不把你训练成算法研究员，而是帮助你掌握：

- 大模型应用后端的核心范式。
- 面试中必须能讲清楚的概念。
- 工作中真实会遇到的工程场景。
- Go 后端如何承接模型调用、流式输出、RAG、工具调用、MCP、沙盒和文件管理。
- 如何通过考题、编程题和项目验收判断自己是否真正掌握。

## 使用方式

建议每次只学习一章。学完一章后，需要完成该章末尾的“本章完成标记”。

每章都按统一结构展开：

1. 学习什么。
2. 为什么学习。
3. 知识详细内容。
4. 考题、编程题与验收。
5. 和其他知识点的相关性。
6. 演变过程与后续方向。
7. 工作中典型场景。
